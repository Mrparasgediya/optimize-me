Instagram image server

Routes

  • POST: /image/upload to upload image (valid image types jpg, png, webp, jpeg)
  • GET: /image/:imageId to get image
    query parameters
    • h= height of image max value= imageheight
    • w= width of image max value= imageheight
    • format= format of image,value = webp, jpg, jpeg, png, auto (display image according browser support) default value = uploaded image type
    • q= quality of image between 0 to 100(default)
    • fit= position of image after resized. values are= cover(default), contain, fill, inside, outside
    • position= position of image after resized. values are= top, right-top, right, right-bottom, bottom, left-bottom, left, left-top. tip: better use on fit cover or contain
    • b= blur of image between 0.3 to 1000
    • brightness= brightness of image between 0 to 100
    • saturation= staturation of image between 0 to 239
    • hue= Degrees for hue rotation
    • tint= tint image format= (r: redValue, g:greenValue, b: blueValue)
    • grayscale | greyscale = apply grayscale effect to image. value= true or false
  • PUT: /image/:imageId to update image
  • DELETE: /image/:imageId to delete image
note: to update or upload image keyname of image must be "image"
